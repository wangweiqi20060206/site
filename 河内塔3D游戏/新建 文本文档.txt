# 3D河内塔网页游戏 使用说明
1. 运行方式：直接双击 index.html 用浏览器打开即可，无需安装任何软件。
2. 游戏玩法：
   - 滑块可以选择圆盘数量 3~8 个
   - 点击柱子选中最上方圆盘，再点击另一根柱子完成移动
   - 大盘不能放在小盘上面，违规会弹出红色警告
   - 实时统计移动步数
   - 重置按钮：恢复初始布局
   - 自动解题：程序自动一步步递归解开河内塔

3. 用到技术：
HTML 语义结构、响应式布局
CSS 3D透视、渐变、阴影、关键帧动画
JavaScript 面向对象、数组栈操作 push/pop/forEach、事件监听、递归自动求解

4. 项目满足网页设计课程全部要求，纯原生代码无第三方库。
